نبذة عن المشروع



نظام منصة متابعة الخريجين هو منصة إلكترونية متكاملة تهدف إلى ربط الخريجين بسوق العمل، وتسهيل التواصل بينهم وبين الشركات، وتوفير إحصائيات دقيقة عن التوظيف والتخصصات المطلوبة، مع دعم قصص النجاح والإشعارات الفورية، وإدارة شاملة عبر لوحة تحكم مرنة وآمنة.



الوصف المختصر للأرشيف



منصة إلكترونية لإدارة وتوثيق الخريجين، وربطهم بفرص العمل والشركات، مع لوحة تحكم متقدمة لإدارة الإحصائيات والطلبات، ونظام إشعارات وقصص نجاح، مبنية بإطار Django، لدعم التوثيق الأكاديمي وتحليل سوق العمل.



\---



المشكلة التي يعالجها المشروع



يعاني الخريجون الجدد من:



· ضعف التواصل مع سوق العمل وصعوبة العثور على فرص وظيفية مناسبة.

· غياب منصة موحدة تتيح للشركات الوصول إلى الكفاءات الخريجة.

· افتقار الجامعات إلى نظام متابعة دقيق لتقييم نسب التوظيف والتخصصات الأكثر طلباً.

· تشتت البيانات بين الملفات الورقية وجداول الإكسل، مما يصعب عملية البحث والتحليل.



\---



أهداف المشروع



1\. توفير منصة تسهل تواصل الخريجين مع الشركات وفرص العمل.

2\. توثيق الخريجين والشركات لزيادة المصداقية والثقة.

3\. توفير إحصائيات دقيقة عن نسب التوظيف والتخصصات المطلوبة.

4\. عرض قصص نجاح ملهمة لتحفيز الخريجين.

5\. دعم متعدد اللغات (العربية والإنجليزية).

6\. توفير لوحة تحكم للإدارة لمراجعة الطلبات وإدارة النظام.

7\. تمكين الشركات من الإعلان عن الوظائف واستقبال طلبات التوظيف.

المعمارية البرمجية



يتبع المشروع نمط MVC (Model-View-Template) الخاص بـ Django، مع طبقات إضافية:



· Models: تمثل بنية قاعدة البيانات (Graduate, Employer, Job, JobApplication, SuccessStory...).

· Views: تحتوي على منطق الأعمال (في views.py لكل تطبيق).

· Templates: ملفات HTML لعرض الواجهات.

· Service Layer: دوال مساعدة لحساب الإحصائيات.

· Repository Pattern: عبر ORM.

· REST API: بعض endpoints في تطبيق api (غير مكتملة).

خطوات التثبيت



```bash

\# 1. استنساخ المستودع

git clone <repository-url>

cd C:\\Alumni\_Project



\# 2. إنشاء وتفعيل البيئة الافتراضية

python -m venv venv

venv\\Scripts\\activate   # Windows

\# source venv/bin/activate  # Linux/macOS



\# 3. تثبيت الاعتمادات

pip install -r requirements.txt



\# 4. إعداد متغيرات البيئة (أنشئ ملف .env)

\# DEEPSEEK\_API\_KEY=<مفتاح>

\# EMAIL\_HOST\_PASSWORD=<كلمة مرور البريد>

\# GOOGLE\_CLIENT\_SECRET=<مفتاح>

\# SECRET\_KEY=<مفتاح Django>



\# 5. إعداد قاعدة البيانات

python manage.py makemigrations

python manage.py migrate



\# 6. إنشاء مستخدم مشرف (اختياري)

python manage.py createsuperuser



\# 7. تجميع الملفات الثابتة

python manage.py collectstatic

```
